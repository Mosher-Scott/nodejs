Math Calculator

This page contains a single math calculator. The user will enter two numbers, and then pick an operation to be performed on those numbers.

As a side note, this form does NOT include validations on the user inputted values.

Once the user clicks "Submit", they will be taken to a new page that displays the total. The user can also click on the "Ajax math!" button, and that will display the total on the same page.
